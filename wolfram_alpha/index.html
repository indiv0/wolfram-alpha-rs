A library providing Rust bindings for the Wolfram|Alpha web API.

The library provides a WolframAlphaRequestSender trait which can be implemented by various request senders. These implementations may then be used to execute requests to the API.

Response bodies are deserialized from XML into structs via the serde_xml library.
